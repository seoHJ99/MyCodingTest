패킷 교환 방식
1. 동적인 대역폭
2. 패킷단위로 분할
3. 배달될때까지 메시지 저장
4. 패킷마다 별도의 전송로
5. 배달 불가하면 알려줌
6. 각 패킷마다 오버헤드 비트

/etc/host : 소규모 lan 환경에서 dns 처럼 작동. 1순위
/etc/resolv.conf : 외부 인터넷과 연결될때 dns.

virtualBox : 리눅스 가상화 기술. VT-x와 AMD-V 를 기반으로 전가상화를 지원
XVM : QEMU 라는 CPU 에뮬레이터. CPU 전가상화.
XEN : 리눅스 cpu 반가상화 기술. 

리눅스 클러스터
1. HPC - 고계산용. 즉, 베어울프 클러스터. 병렬구조.
2. LVS - 부하분산. 즉, 로드밸런싱
3. HA - 고가용성. 즉, 백업


Ansible : 인프라를 코드 기반으로 자동화하는 도구. 소프트웨어 프로비저닝 관리. 
Tizen : 삼성과 인텔이 주도해서 만든 리눅스 기반 모바일 운영체제.
GENIVI : 리눅스 커널 기반 자동차용 플렛폼.
MeeGo IVI : 모바일 운영체제
Android IVI : 차량 내부 운영체제.
Rocky Linux : 그레고리 커터가 주도. CentOS 의 정신적 후속작.
SSL/TLS 프로토콜 : 응용 계층과 전송 계층 사이의 독립적 계층. OSI 7계층 바깥
QNX : 유닉스계열 실시간 운영체제. 임베디드에서 쓰임. 리눅스스 커널 기반이 아님.
YaST : 리눅스 설정 및 구성 도구
아두이노 : 이탈리아의 IDI 개발. 단일 보드 마이크로 컨트롤러.

LVM :로지컬 볼륨 매니지먼트. 디스크를 동적으로 관리할수 있는 리눅스 기능.
    PV : 실제 물리적 저장장치. 디스크 내의 파티션들을 LVM 에서 사용하기 위해서 PV로 초기화 해야함.
    PE : pv 를 구성하는 블록. 기본 크기 4mb.
    VG : 물려적 볼륨의 모음.
    LV : 논리 볼룸. LVM 의 디스크 파티션. 
    LE : 논리 블록. 기본 크기 4mb
    FS : 파일 시스템. 논리적 볼륨 위에 생성되며 파일 관리 시스템


RAID-6 용량 계산 : 디스크 숫자 -2 * 디스크 용량

[컨트롤 + \] : SIGQUIT 전송. 3번
[컨트롤 + c] : SIGINT. 2번
[컨트롤 + z] : SIGTSTP. 20번. 포어그라운드 작업을 백그라운드로 전환. 또는 현재 작업중인 프로세스 일시 정지
$MAIL : 메일 관련 경로가 저장된 환경변수
