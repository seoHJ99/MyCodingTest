개요

지금까지 여러분은 애플리케이션의 모든 구성 요소를 생성했으며 웹에서 그러한 구성 요소에 액세스하도록 허용하는 방법을 알아보았습니다. 나머지 함수에 대한 API 메서드를 더 추가하기 전에 액세스 보안을 설정할 수 있어야 합니다. 이 실습에서는 사용자를 인증하고 API 사용 권한을 부여하도록 Amazon Cognito를 설정하는 방법에 대해 알아봅니다. 그런 다음, 모든 기능을 갖춘 애플리케이션을 완료하도록 프런트엔드 웹 애플리케이션을 구성합니다.

실습을 완료하면 배포는 다음 다이어그램과 같이 됩니다.

목표

이 실습을 완료하면 다음을 할 수 있게 됩니다.

  • Amazon Cognito를 사용하여 웹 애플리케이션을 위한 사용자 풀과 앱 클라이언트를 생성합니다.
  • 새 사용자를 추가하고 Amazon Cognito CLI를 사용하여 로그인할 수 있는지 확인합니다.
  • Amazon Cognito를 권한 부여자로 사용하도록 API Gateway 메서드를 구성합니다.
  • API 호출 중에 JWT 인증 토큰이 생성되는지 확인합니다.
  • Swagger 가져오기 전략을 사용하여 API Gateway 리소스를 신속하게 개발합니다.
  • Amazon Cognito 및 API Gateway 구성을 사용하도록 웹 애플리케이션 프런트엔드를 설정하고 전체 애플리케이션 기능을 확인합니다.
