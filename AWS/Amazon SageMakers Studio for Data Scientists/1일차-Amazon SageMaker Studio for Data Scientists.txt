파라미터 : 훈련하는 도중에 필요한 파라미터. 가중치 등
하이퍼파라미터 : 훈련하기 전에 필요한 파라미터. 훈련 횟수, 데이터 범위 등.

회귀 : 여러 개중 하나의 값으로 확정
분류 : 둘중 하나로 분류
배치 : 일괄 처리

모델 개발의 단계
데이터 집합 준비 -> 알고리즘 선택 -> 모델 훈련 -> 모델 평가 및 튜닝(하이퍼파라미터 튜닝)

훈련, 검증은 동일한 분포의 데이터 필요. 즉, 훈련에서 8:2였다면 검증도 8:2여야 함.
aws 에서는 알고리즘을 컨테이너로 만들어 ECR 에 저장하여 제공함.

앙상블 : 여러개의 결과 묶어서 더 좋은 성능을 내는것.
Bagging : 앙상블의 한 종류. 다른 결과가 현재 데이터에 영향을 미치지 않음
Boosting : 앙상블의 한 종류. 이전 결과가 현재 데이터의 가중치에 영향을 미치는 것.

혼동 행렬: 분류할때 사용하는 행렬.
세로열이 실제, 가로열이 예측치를 의미.
------------------------------
            예측
실  (참 긍정)   (거짓 부정) 
제  (거짓 긍정) (참 부정)
--------------------------------

첫 값이 참, 즉, 실제값이 참인 값들은 분류가 옳바르게 됨을 의미.
(실제 값이 참인 개수) / (전체 값) = 정확도

정밀도 - 얼마나 잘 맞추는가? - 즉, 얼마나 틀린 예측을 조금 하는가?
(예측치와 실제값이 모두 긍정인 개수) / (예측치가 긍정인 개수) = 정밀도

재현율 - 얼마나 잘 예측하는가? 즉, 예상치가 얼마나 잘 들어맞는가?
(실제값과 예측값이 모두 참, 긍정인 개수) / (실제값이 참인 개수) = 재현율

수신기 작동 곡선 ROC
참 긍정 TP(모델이 실제로 적격인 값을 적격으로 분류하는 빈도)
거짓 긍정 FP(모델이 실제로 적격인 값을 거짓으로 분류하는 빈도)
참 긍정이 높을수록 좋음. 
해당 그래프를 그렸을때, 그래프 아래값을 AUC 라고 부름.
AUC 가 높을수록 좋음.


회귀 문제 지표.
예측치와 실제 값의 차이 제곱을 어쩌구 한것이 평균 제곱 오차(MSE)
R제곱 : 높을수록 좋은 모델. 그러나 특성 개수가 많으면 나쁜 모델인데, 많으면 많을수록 R제곱 값이 올라가기에 이런 영향을 받지 않기 위해 '조정된 R제곱' 이란 값을 쓴다.
