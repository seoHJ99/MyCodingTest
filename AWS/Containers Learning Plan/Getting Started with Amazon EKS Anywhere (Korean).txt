Amazon EKS Anywhere
아마존 EKS 용 오픈 소스 온프레미스 배포 옵션.
온프레미스 환경에서 AWS 의 클라우드의 쿠버네티스와 일치하는 쿠버네티스 사용 가능.
aws 계정 없이도 저액 요금으로 사용 가능.
클라우드 기반 솔루션이 아님. 클라우드로 변경하기 전 클라우드 같은 환경을 온프레미스에서 먼저 사용하는데 의의.
Amazon EKS Connector 를 통해 AWS EKS Cloud 콘솔에 연결 가능. 콘솔은 가시성 제공. 운영 기능x.
전체 클러스터를 구성하는데 사용 할 수 있는 단일 구성 파일 제공.
GitOps 기능으로 클러스터 수명 주기 관리.

기본 기술적 개념 6가지
1. Amazon EKS Connector
2. 큐레이티된 패키지 : 소스 기반으로 만들어진 컨테이너 패키지. Amazon 인증을 받아서 서명 이미지를 제공함.
3. 클러스터 구성 : YAML 파일. 클러스터 구성 정보 포함된 도메인 객체
4. 패키지 컨트롤러 : 패키지를 설치, 업그렝드, 삭제 등이 가능. 지속적으로 모니터링하여 새 패키지 다운
5. GitOps : 버전 제어 방식으로 클러스터를 관리할수 있도록 함.
6. IRSA : IAM 역할. AWS 서비스에 대한 액세스 권한을 제공하는 IAM 역할 생성 가능.
