기계학습이란?
데이터를 활용하여 컴퓨터를 학습시키는 작업
Label : 종속변수 - 알고자 하는 값
Feature : 독립변수 - 결과에 영향을 미치는 변수
Weight : 가중치 - x에 곱해지는 값. 독립변수의 영향의 정도를 줄이거나 늘림
Bias : 편향 - 오차값. 가중치와 독립변수로 설명되지 않는 부분을 보충치
Label = Weight * Feature + Bias

기계학습의 문제
회귀 : 수치를 예측하는 문제
분류 : yes or no 같은 이지선다 문제를 예측하는 문제

기계학습의 유형
지도 : 인간이 직접 컴퓨터를 학습시킴.
    - 양질의 레이블이 지정된 데이터가 필요
    - 선형학습자, KNN 등의 알고리즘이 존재.
비지도 : 사람의 지도 없이 기계가 알아서 추측. 인간도 데이터의 의미를 모를때 사용.

강화 : 목표지향적인 학습. Agent 가 알아서 상황에 맞는 Action 을 하고, 그에 맞는 보상을 주는 방식.

모델의 과적합(오버피팅)
너무 많은 데이터를 학습시켜 특정 경우에서만 옳바르게 작동함.
데이터 분할 : Training, Validation, Test 로 데이터를 분할. Training 과 Validation 이 훈련에 사용됨.

손실함수
|예측값 - 실제값|
손실이 최소화되도록 파라미터 조정 필요. 최적화.
경사하강법 : 함수의 기울기를 이용하여 손실값의 최소값을 찾는 방법.
학습률이 너무 작으면 시간이 부족, 너무 크면 최저점을 못찾음.

기계학습의 파이프라인
비지니스 문제 -> 문제 공식화 -> 데이터 수집 및 전처리 -> 모델 훈련 및 튜닝 -> 데이터 보충 -> 모델 평가
