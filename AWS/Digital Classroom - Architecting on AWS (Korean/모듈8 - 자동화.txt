AWS CloudFormation

IaC 는 인프라를 코드로 다룸
인프라를 코드로 다루기에 복제, 재배포, 롤백 등이 편해짐

CloudFormation : IaC 템플릿을 저장, 요청할수 잇는 서비스
    - 인프라 템플릿은 json 이나 yaml 로 작성됨
    - 이전 버전으로 롤백도 가능

스택 : 단일 단위로 관리되는 aws 리소스 모음.
    - 여러 리소스를 하나의 단위로 배포 및 관리 가능
    - 중첩 스택 지원. 어플리케이션을 모듈로 나눠 서로 독립적으로 사용 가능
    - 아키텍처를 계층적으로 나눠 독립적으로 관리 가능


----------------------------------
인프라 관리

인프라 도구
AWS Elastic Beanstalk
-코드를 전달하면 Beanstalk 가 알아서 어플이 돌아갈 인프라를 생성함.
-도커도 지원

aws 솔루션 라이브러리
   - 사전에 구축된 참조 아키텍처를 제공
   - aws 아키텍트가 승인한 솔루션

AWS CDK
프로그래밍 언어를 이용하여 템플릿을 생성, 이를 CloudFormation에 전달 가능.

System Manager
-리소스 그룹의 리소스 변경이나 모니터링, 운영 정보, 알림 등을 확인할수 잇다.
-aws 와 멀티클라우드에 대한 중심 위치를 제공한다.
-제어는 못하고 오직 관리에 초점.
- 스택의 계층, 어플 등 리소스의 논리적 그룹 생성
