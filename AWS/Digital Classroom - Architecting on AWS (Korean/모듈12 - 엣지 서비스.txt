Amazon Route53

가용성과 확장성을 모두 갖춘 aws 용 dns 서비스
퍼블릭 호스팅 영역과 프라이빗 호스팅 영역으로 나뉨
프라이빗으로 만들어서 vpc 에 연결하면 vpc 내에서만 도메인 이름이 존재
가중치로 한쪽에 트래픽이 몰리게하거나 지리적 위치 기반 라우팅 가능

1. 단순 라우팅

2. 장애조치 라우팅
    상태확인 하고 비활성화 상태면 대기 vpc로 보냄

3. 가중치 기반 라우팅
   한쪽에 더 많이 몰아줌

4. 지리적 위치 라우팅
   사용자의 지리적 위치에 맞춰 가까운 리전에 라우팅. 리전별 연결도 가능.

5. 지연시간 기반 라우팅
   지연시간이 가장 짧은 리전으로 보내는 것도 가능

6. 다중값 응답 라우팅
    라우팅 테이블에서 무작위로 골라 보내는 방법

Amazon CloudFront

다른 리전에서도 빠르게 콘텐츠를 볼수 있게 엣지 로케이션에 미리 캐싱된 콘텐츠를 보여줌.
이게 cdn 이고 aws 의 cdn 서비스가 Amazon CloudFront임.

오리진(원본)에 가해지는 부하가 줄어듬

DDoS 방어

aws shield
 standard 는 기본적으로 제공되며 3,4계층까지는 방여해줌
 advanced 는 추가 결제. 애플리케이션을 보호하는 기능 제공

waf 는 계층 7에서 작동하며 CloudFront와 appsync 등과 통합
waf 는 acl로 구성됨
sql 인젝션 방어, 특정 경로에 대한 공격 방어도 가능
acl 규칙문을 만들수도 있음
방어 규칙(템플릿) 을 제공

firewall Manager
방화벽의 중심위치. 방화벽 및 네트워킹과 관련된 모든 것을 시각화 가능
